# 💻 Competitive Programming Training – Inatel

Repositório dedicado às atividades e exercícios realizados durante o treinamento da equipe de **programação competitiva do Inatel**.

## 🚀 Objetivo

Este repositório tem como objetivo:

* Praticar lógica de programação
* Aprimorar habilidades em algoritmos e estruturas de dados
* Resolver problemas clássicos de programação competitiva
* Criar consistência nos estudos

## 🧠 Conteúdos abordados

* 🔢 Matemática básica para programação
* 📊 Estruturas de dados
* 🔍 Busca e ordenação
* 🔁 Recursão
* ⚡ Programação dinâmica
* 🌐 Grafos

## 🛠️ Linguagem utilizada

* C++

Cada pasta contém:

* Código-fonte das soluções
* Possíveis comentários explicando a lógica

## 🏆 Plataformas utilizadas

* Beecrowd (URI)


## 📌 Observações

Este repositório é focado em aprendizado e evolução contínua.
As soluções podem não ser sempre as mais otimizadas inicialmente, mas refletem o processo de melhoria ao longo do tempo.
